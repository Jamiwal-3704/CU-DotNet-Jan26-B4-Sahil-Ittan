# Capgemini Training Repository

This repository contains all the work completed during my  **Capgemini Training Program** .
It is organized into three main sections to clearly separate **assignments, assessments, and mini projects** completed throughout the training.

## Repository Structure

```
Capgemini-Training
│
├── Assignments
│   └── Contains daily exercises, coding tasks, and practice problems completed during the training.
│
├── Assessments
│   └── Contains weekly assessments and evaluation tasks given during the training program.
│
└── MiniProjects
    └── Contains mini projects developed as part of the training to apply learned concepts.
```

## Folder Details

### Assignments

This folder includes all **daily assignments and exercises** completed during the training sessions.
These tasks helped in strengthening concepts and improving coding skills.

### Assessments

This folder contains **weekly assessments** conducted to evaluate understanding of the topics covered during the training.

### MiniProjects

This folder contains **mini projects** developed during the training to demonstrate practical implementation of the concepts learned.

## Purpose of This Repository

The purpose of this repository is to:

* Track progress during the **Capgemini training program**
* Organize all assignments, assessments, and projects in a structured way
* Maintain a record of learning and development throughout the training
