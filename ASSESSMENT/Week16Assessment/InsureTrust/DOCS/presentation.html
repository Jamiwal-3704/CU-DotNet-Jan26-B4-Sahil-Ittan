Claim, Support & Admin Service

:7004 ClaimDB Tushar

Claims

ClaimNumber (CLM4001), Description, MaturityAmount, Status (Pending/Approved/Denied), up to 12 document uploads. Approval credits user balance.

TicketNumber (SUP3001), Subject, Description, Status (Pending → InProgress → Resolved), optional attachment, admin response.

Aggregated stats: total users, active policies, pending claims, open tickets, total revenue. User list with policy counts.
